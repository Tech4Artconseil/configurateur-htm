Visionneuse Fauteuil — Panneau d'information
=============================================

Ce fichier sert de contenu pour le panneau "i" (Afficher / cacher informations).
Il est chargé par défaut depuis la racine : `left-panel.txt`.

NOTE : l'application affiche actuellement le contenu en tant que texte brut (préformaté)
via un `<pre>` (le contenu est inséré avec `textContent`). Si vous souhaitez du HTML
interprété, il faudra modifier `app.js` pour utiliser `innerHTML` (avec nettoyage/sanitation).

-------------------------

1) Exemple simple (paragraphe)

La visionneuse permet de charger un modèle GLB/Gltf et d'appliquer des textures PBR.
Utilisez le bouton "i" pour ouvrir ce panneau.

Conseils pratiques
- Placez ce fichier `left-panel.txt` à la racine du projet (même dossier que `index.html`).
- Encodage recommandé : UTF-8 (pour accents et symboles)
- Si vous préférez HTML rendu, changez la ligne
  `pre.textContent = txt;` en `contentDiv.innerHTML = sanitize(txt);`
  (ajoutez une fonction `sanitize` ou utilisez une librairie de sanitation pour éviter les XSS).

Exemples supplémentaires
- Vous pouvez copier-coller ci-dessus une section Markdown pour la modifier rapidement.
- Pour de la documentation riche, créez `docs/notice.md` et appelez
  `window.loadLeftTextFromUrl('docs/notice.md');`
